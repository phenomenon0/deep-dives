GAME CHARACTERS

One lightweight browser sandbox. One hand-built chibi mascot. Movement, jump tuning, collision mercy, attack punctuation, and click-to-move navigation taught on the same body._

This page is character-first on purpose. Not “here are fifty engine subsystems.” Not “here is generic AI content.” One small hero, one camera, one controller, and a sequence of chapter modes that show how game feel is authored. If the body feels dead, the game feels dead. If the body feels readable, generous, and slightly overcommitted in the right places, players forgive almost everything else.

// Shared Sandbox

The lab below is the whole lesson. Same scene. Same renderer. Same original mascot. Same controller. Chapter buttons only change the room, the teaching overlay, and which numbers you are encouraged to touch. That is how real character work happens: not by writing a perfect spec, but by pushing on one living thing until it starts to answer back.

What To Watch

Look at stop distance, turn commitment, body lean, and how quickly the pose agrees with the input.

// 01 Move Like You Mean It

Players do not feel speed directly. They feel how quickly the body begins, how stubbornly it turns, and how long it takes to stop. That is why two characters with the same top speed can feel like different genres. Mario 64 is readable because you can predict the negotiation between intention and momentum within a few frames.

js// speed is the destination; acceleration is the feeling
const targetVX = wishX * tuning.speed;
const targetVZ = wishZ * tuning.speed;
velocity.x = approach(velocity.x, targetVX, tuning.accel * dt);
velocity.z = approach(velocity.z, targetVZ, tuning.accel * dt);
Try This

Stay in Move mode. Leave `speed` around `6.0`. Drag `accel` down near `12` and then back up near `42`. Same room. Same camera. Same top speed. Different game.

// 02 Jump Arcs Are Authored, Not Discovered

Jump feel is not “whatever gravity gives us.” It is a promise about where the body will go if the player commits now. Celeste is the clean teaching example: coyote time, input buffering, and a deliberately shaped fall curve are not cheats against the player. They are respect for the player’s reading of the scene.

Coyote Time

Keep jump legal for a beat after the feet technically left the ground.

Input Buffer

Remember an early jump press so landing still pays it out.

Asymmetric Gravity

Let ascent and descent feel different even in a tiny controller.

jsif (input.jumpPressed) state.jumpBuffer = 0.14;
if (state.grounded) state.coyote = 0.10;

if (state.jumpBuffer > 0 && state.coyote > 0) {
  velocity.y = tuning.jump;
  state.grounded = false;
}
Try This

In Jump mode, cut `gravity` until the body feels moon-bound. Then restore gravity and lower `jump` instead. One changes hang time. The other changes commitment.

// 03 Flow Is The Body Agreeing With The Input

Animation is not a reward after movement. It is part of movement. Lean, anticipation, recovery, stride width, and landing squash are the visual proof that the input mattered. Without that proof, the controller can be numerically fine and still feel emotionally dead.

jsconst runPhase = time * (3.2 + speed01 * 8.2);
rig.legL.rotation.x = Math.sin(runPhase) * stride;
rig.legR.rotation.x = Math.sin(runPhase + Math.PI) * stride;
rig.hips.rotation.z = -turnLean * 0.18;
rig.root.scale.set(1 + landSquash, 1 - landSquash, 1 + landSquash);
Watch The Pose, Not Just The Feet

Flow mode is where you feel the difference between “moving geometry” and “a little cartoon body making a decision.” Zig-zag and watch the cap, torso, and sword hand agree with the turn.

// 04 Collision Is Mercy Written In Code

Character collision exists to preserve intention when geometry gets petty. The body is not a court stenographer. It is a special controller that helps the player keep their sentence through steps, ramps, edges, and shallow walls. Good collision feels honest because it quietly cheats in the player’s favor.

jsif (colliding && stepTop - feetY <= tuning.step && grounded) {
  position.y = stepTop;
} else {
  // keep the round body from snagging on corners
  position.x += pushX;
  position.z += pushZ;
}
Try This

In Collision mode, drop `step` almost to zero and try the tiny staircase. Then raise it back above `0.4`. That one number is the difference between “hostile” and “polished.”

// 05 Action Feel Is Pause, Flash, Knockback, Sound

A sword swing is not one event. It is anticipation, contact, hitstop, recoil, knockback, and an audio punctuation folded so tightly they read as one hit. Smash is the clean historic teacher here: consequence becomes visible because the game briefly lies about time.

jsif (hitDummy) {
  state.hitstop = tuning.hitstop;
  dummy.velocity.add(forward.multiplyScalar(tuning.knockback));
  spawnImpactRing();
  playBlip(180, 0.08, 'square');
}
Try This

In Action mode, set `hitstop` to `0` and slash the dummy. Then bring it back to `0.07`. Almost no math changed. The feeling changed immediately.

// 06 Navigation Is Just Another Input Source

Click-to-move should not feel like a second game glued on top of the first. The pathfinder picks a route, but the body still has to turn, accelerate, and stop like the same character. Good navigation is not “look how smart the search is.” It is “that still looks like the same little hero.”

jsconst path = aStar(startCell, goalCell, grid);
const next = path[index];
const wish = next.clone().sub(position).setY(0).normalize();
velocity.x = approach(velocity.x, wish.x * tuning.speed, tuning.accel * dt);
Try It With Your Hands

Click a target in Navigation mode, let the hero begin walking, then touch `WASD`. Manual input cancels autopilot. That is the point: navigation is another input source to the same controller, not a replacement controller.

// 07 The Tiny Game

Final mode stops introducing the pieces and starts making them cooperate. Collect the crystals. Jump to the raised platforms. Slash the blocker if it is in your lane. Click a destination if you want the path layer to carry the body. Reach the gate after collecting all three crystals. That is enough to prove you have a character, not just a bag of tricks.
